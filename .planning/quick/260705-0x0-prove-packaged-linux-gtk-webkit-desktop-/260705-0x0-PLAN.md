# Quick Task 260705-0x0: Desktop Native Runtime UAT

## Goal

Prove the packaged Linux GTK/WebKit Desktop Native shell can be driven as the app runtime, then record the honest
transport result from the native engine.

## Tasks

1. Add WebKitGTK automation support to `packaging/desktop/meshdrop-desktop.c`.
   - Accept `--automation`.
   - Enable `WebKitWebContext` automation.
   - Return automation-controlled `WebKitWebView` instances for WebDriver sessions.

2. Add a focused native runtime smoke.
   - Build and unpack `meshdrop-desktop-linux`.
   - Launch WebKitWebDriver against the packaged `bin/meshdrop-desktop`.
   - Prove the shell injects the Desktop Native target manifest.
   - Prove the runtime gates WebRTC off when the native engine does not expose `RTCPeerConnection`.

3. Update Desktop Native status/runbook.
   - Record source artifact transfer proof separately from packaged native runtime proof.
   - Keep native transfer UAT and signed installer as remaining Desktop Native gaps.
