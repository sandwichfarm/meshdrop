# MeshDrop StartOS Package

This is the StartOS package source scaffold for MeshDrop.

- Keep `README.md` and `instructions.md` aligned with package behavior.
- Keep `docs/uat/start9.md` in the upstream MeshDrop repo aligned with package behavior.
- Do not call this target complete until `make` produces an `.s9pk`, install UAT passes on a StartOS device, and a file
  transfer is initiated successfully from the device.
